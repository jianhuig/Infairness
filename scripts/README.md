# Scripts

This folder contains simulation and exploratory analysis scripts used during package development.

These files are not part of the package API and are excluded from package builds through `.Rbuildignore`.

- `RunSimParallel.R`: parallel simulation driver
- `TwoStep.R`: helper functions for the two-step simulation workflow
- `add_cov.R`: simulation workflow focused on additional covariates
- `stepII.R`: second-stage estimation helper
